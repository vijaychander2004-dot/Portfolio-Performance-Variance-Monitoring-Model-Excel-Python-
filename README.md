# Portfolio Performance & Variance Monitoring Model (Excel + Python)

## Project Overview
This project builds a structured portfolio monitoring framework to analyze portfolio performance, track allocation impact, and evaluate variance between expected and realized returns.

The model integrates Excel-based portfolio calculations with Python-driven financial analytics and simulations.

It helps understand how different stock allocations affect overall portfolio return and risk.

---

## Key Features
- Portfolio return calculation using weighted allocations
- Variance comparison between expected and actual portfolio returns
- Portfolio performance monitoring
- Monte Carlo simulation for potential future price movements
- Basic technical analysis for trend evaluation
- Allocation consistency checks to validate portfolio weights
